# Proyecto-7
Herramientas de desarrollo de software.

-Investigar y analizar herramientas de desarrollo ampliamente utilizadas en la industria
-Comprender el propósito y las buenas prácticas para cada herramienta
-Mostrar cómo estas herramientas se integran en un flujo de trabajo típico
-Evaluar ventajas y limitaciones de cada una
-Demostrar cómo analistas de datos o científicos de datos pueden usar estas herramientas en sus propios proyectos
